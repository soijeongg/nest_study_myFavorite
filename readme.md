### 최애 자랑 프로젝트 myFavorite
nest.js 공부를 위한 프로젝트 
내 최애들을 소개하고 최애의 사진 공유 및 댓글로 소통
웹소켓을 이용한 댓글, 좋아요 시 알람


### 사용 스택
typeScript, nest.js, docker, aws S3, aws ECR, aws EC2
